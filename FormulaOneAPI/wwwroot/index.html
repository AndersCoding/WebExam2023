F1 Documentation

Introduction

This introduction will show you the resources for the Drivers API, and how to properly use if for yourself

REST

The base URL requests go over https

There are currently one available resource

  • Drivers: used to get drivers
{
"Drivers:" http://localhost:5160/Driver
}

Info and pagination

These are the datatypes that the Driver API includes:

Key Type Description
Id int The id of the driver
Name string The name of the driver
Age int The age of the driver
Nationality string Nationality of the driver
Image string An image of the driver
GET http://localhost:5160/Driver
[
{ "id": 0,
"name": "string",
"age": 0,
"nationality": "string",
"image": "string" }
]

Drivers

There are 6 drivers in the Driver database


Get a single driver by ID

You can get a single driver by adding the id to the url: /driver/4

Get http://localhost:5160/Driver/4
{ "id": 4,
"name": "Fernando Alonso",
"age": 43,
"nationality": "Spain",
"image": "fernandoalonso.jpeg"
}
